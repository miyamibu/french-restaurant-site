# ビストロジョワ — フレンチレストランサイト

静的なレストランサイトの雛形です。`index.html` をブラウザで開くだけで閲覧できます。

## 店舗情報

- 店名: ビストロジョワ
- 住所: 埼玉県戸田市中町２－１４－２４
- 電話: 048-441-4998（`tel:0484414998`）
- 営業時間: 11:00–22:00
- 定休日: 月曜日・火曜日（祝日は営業する場合があります）
- お支払い: 現金のみ

## 構成

- ホーム: `index.html`
- メニュー: `menu.html`（A:手軽コース・B:しっかり・C:シェフおまかせ。昼/夜で価格が異なる表記に対応。全て税込）
- 私たちについて: `about.html`
- ご予約: `reservation.html`（現在は電話のみ受付）
- アセット: `assets/css/styles.css`, `assets/js/main.js`, `assets/img/*`
- SEO: `robots.txt`, `sitemap.xml`, `site.webmanifest`

## メニュー（税込）

- A: 手軽コース — 昼 ¥1,250 / 夜 ¥1,750（軽やかに楽しめる入門コース）
- B: しっかり — 昼 ¥1,500 / 夜 ¥2,000（定番の満足コース）
- C: シェフおまかせ — 昼 ¥1,750 / 夜 ¥2,250（その日の最良の食材で）

注: 夜は昼価格に+¥500。仕入れにより内容が変わる場合があります。

## 使い方

1. デスクトップの `french-restaurant-site/index.html` をダブルクリックで開く
2. 文言・価格・住所・電話番号を実際の情報に更新（上記は雛形です）
3. 画像はプレースホルダー（Unsplash/placehold.co）です。公開時は店舗の実写真・ロゴに差し替えてください。

## デプロイ

静的ホスティング（GitHub Pages, Netlify, Vercel, Cloudflare Pages など）に `french-restaurant-site/` フォルダを公開してください。
